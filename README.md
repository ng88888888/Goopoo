# Goopoo 🪿

A pixel art desktop goose that lives in the corner of your screen.

Goopoo wakes up when you open Telegram, plays a random animation, and disappears 
after 15 minutes of inactivity. Double-click him to make him poop.

This is the first project I ever coded. Built with help from Claude, April 2026.

## Animations
- Excited flap when Telegram opens
- Goofy side-to-side dance
- Sleeping with snore Z's
- Drops a poop and waddles away with pride

## How to run
1. Install Python 3.8+
2. `pip3 install PyQt5 Pillow pyobjc-framework-Cocoa`
3. `python3 sprites.py` (first time only)
4. `python3 main.py`

## Controls
- **Double-click** — poop on demand
- **Left-click drag** — move Goopoo around
- **Right-click** — say goodbye and quit
